Google 官方推出 agents-cli，一個 CLI 工具讓 coding agent（Claude Code、Codex 等）可以直接建立、評估、部署 AI agent 去 Google Cloud（Gemini Enterprise Agent Platform）。內置多個 agent skills：ADK 開發、deployment、eval、observability。對 Fafa 直接用 Google Agent Dev Kit + Agent Engine 嘅 workflow 極度相關 — 可以大幅簡化 agent 開發到 production 嘅流程。

browser-use/video-use
★ 12.7k新項目

由 browser-use 團隊（browser-use 本身係最 popular 嘅 browser automation for AI agents 工具）推出嘅 video editing 工具。Drop raw footage 入 folder，同 Claude Code / Codex 講你要咩效果，就出 final.mp4。唔係 frame-by-frame 睇片（太貴），而係轉錄 transcript + 結構化 metadata，同 browser-use 俾 DOM 代替 screenshot 嘅思路一樣慳 token。MIT license，完全 open source。

Robbyant/lingbot-map
★ 8.9k新項目

Feed-forward 3D foundation model，可以從 video / image sequence 即時 reconstruct 3D scene。支援 streaming inference（逐 frame 處理）、KV cache temporal consistency、sliding window 處理 3000+ frames。Apache 2.0 license，有 HuggingFace model weights。對 robotics / 3D vision 場景好有用。

github.com開源 3D foundation model，streaming reconstruction，Apache 2.0
diegosouzapw/OmniRoute
★ 500實用

免費 AI gateway，一個 endpoint 連去 237 個 provider（50+ 免費），支援 Claude Code / Codex / Cursor / Cline / Copilot 等 16+ coding agent。內置 17 種 routing strategy（auto-fallback、cost-optimized、round-robin），RTK + Caveman token compression save 15-95% tokens。對用多個模型嘅 developer 嚟講可以大幅降低 API cost。

github.com免費 AI gateway，237 providers，auto-fallback，慳 token
